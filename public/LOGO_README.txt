This folder should contain your logo.png file.

Copy your Apex Classes logo (logo.png) into this public/ folder.
It is referenced in:
  - components/layout/Navbar.tsx   → displayed in the sticky nav bar
  - components/layout/Footer.tsx   → displayed in the footer (white/inverted)

The image displays at:
  - Navbar: height 52px
  - Footer: height 56px, brightness-0 invert (white version)

Recommended logo specs:
  - Format:     PNG with transparent background
  - Min width:  200px
  - Min height: 100px
